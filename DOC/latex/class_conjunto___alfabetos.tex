\doxysubsection{Descripción detallada}
Representa el conjunto de características y operaciones que se podrá realizar sobre un conjunto de Alfabetos. 

Ofrece la operación de lectura y de listar en la consola los alfabetos miembros en el conjunto. También ofrece las operaciones de añadir alfabetos y borrar alfabetos del conjunto. Dado a que es posible que otra classe quiera acceder a algun alfabeto concreto del conjunto, he creado una operación de consulta que retorna un alfabeto del conjunto identificado con un identificador proporcionado.

Como también voy a necesitar saber si existe o no cierto alfabeto en el conjunt he creado una operacion de consulta que retorna un booleano como true si el alfabeto existe y como falso si no existe

Como voy a tener que acceder a los alfabetos del conjunto voy a implementar el conjunto como un map de strings como identificadores y alfabetos.
